Charabia

Visitez le site http://code.google.com/p/charabia/
pour une aide complete.

Utilisez le menu pour creer des clés partagées avec vos contacts, choisissez pour l'un le mode maitre et pour l'autre le mode esclave et scannez les QR-codes de chaque téléphone.

Envoyez des sms sécurisés a vos contacts.
