CharabiaActivity

Sur l'écran précédent vous pouvez:

Dessinez un point d'interrogation pour obtenir de l'aide
Dessinez un S pour envoyer le message
Dessinez un + pour ajouter un destinataire au message
Dessinez un alpha pour effacer tous les destinataires du message
Dessinez un O pour quitter l'application
Dessinez un M pour ajouter un contact partagé sécurisé

Appuyez sur la touche back pour revenir à l'écran précédent.
